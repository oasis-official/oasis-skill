# OASIS Framework

엔터프라이즈 시스템을 구축하기 위해 필요한 필수 구성요소와 업무 절차를 표현한 다이어그램을 실행시켜주는 실행엔진을 포함하는 프레임워크입니다.

# 오아시스는

- 모든 업무 프로세스는 표준화된 업무 표현 방법인 BPMN으로 표현합니다.
- BPMN은 글로벌 표준이므로 세계 어느 곳에서든 소통이 가능합니다.
- BPMN으로 표현한 업무 절차를 오아시스 프레임워크가 직접 실행을 하여, 업무 프로세스 설계자와 개발자 소통 간 발생하는 누락, 착오 등이 없습니다.
- 설계자와 개발자간 원활한 소통은 곧 소프트웨어 품질향상으로 이어집니다.
- 시스템 구축에 필요한 주요 공통기능을 제공하여 개발자가 고객의 요구사항 구현에 집중할 수 있도록 합니다.

# 오아이스 프레임워크 소개

표준화된 개발

 - BPMN 을 이용한 프로세스 개발

 - 최소 코드 작성 유도

편리한 시스템 관리

 - 구조화된 로그 발생으로 추척 용이

 - 재배포 없이 프로세스 변경 가능

 - git 으로 효율적인 자원 관리

독립성

 - OS, 미들웨어에 독립적

 - POJO 사용으로 비즈니스 로직이 프레임워크에 독립적

 - 개발된 모든 로직은 개별로 쉽게 테스트 가능
