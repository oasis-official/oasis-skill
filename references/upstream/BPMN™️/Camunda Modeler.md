# Camunda Modeler

Camunda 사에서 개발한 BPMN 오픈 소스 모델러이다. Windows, Linux, Mac 운영체제에 맞는 설치형 모델러를 제공하고 OASIS 와 잘 연동된다.

![Untitled](Camunda%20Modeler/Untitled.png)

# 다운로드 및 설치방법

Camunda 사의 [다운로드 페이지](https://camunda.com/download/modeler/)에서 설치 환경에 맞는 설치파일을 다운받아 설치한다.

## 요구사항

- Java Runtime Environment 1.8+

### 설치 여부 확인 방법

터미널에서 아래와 같이 입력해서 버전을 확인한다.

```
java -version
```
