# Sequential vs Parallel Multi instance

Sequential multi-instance 태스크는 태스크가 하나씩 실행된다. 한 태스크가 종료되면 그 다음 태스크를 이어 수행하는 것이다.

![Untitled](Sequential%20vs%20Parallel%20Multi%20instance/Untitled.png)

반면 Parallel multi-instance 는 모든 태스크가 동시에 실행된다.

![Untitled](Sequential%20vs%20Parallel%20Multi%20instance/Untitled%201.png)
